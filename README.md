<h4 align="center">Deployed on <a href="http://thapar.edu">thapar.edu</a></h4>
<br />

## Technologies and Libraries

- tensorflow
- flask
- flask_ngrok
- tflearn
- pickle
- sqlAlchemy
- NLTK
- numpy
- profanity_check
- flask_cors

## Endpoints

- /signin
- /email
- /chat
- /email_database
- /chat_database
